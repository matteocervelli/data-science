Visualization Types

1. Time Series Analysis

Purpose: Analyze sales trends over time with daily, monthly, quarterly, and yearly aggregations.

Key Insights: Identifies growth patterns, seasonal trends, and anomalies in sales data.

2. Product Performance

Purpose: Compare performance across different product categories and sales channels.

Key Metrics: Revenue by product type, profit margins, channel effectiveness.

3. Geographic Analysis

Purpose: Understand regional sales patterns and country-level performance.

Coverage: Analysis across all regions and countries in the dataset.

4. Correlation Heatmap

Purpose: Identify relationships between features for ML model development.

Top Correlations with Revenue:

  • Total Cost: 0.985

5. Executive Dashboard

Purpose: High-level KPI overview for stakeholders.

Includes: Revenue trends, profit margins, order volumes, regional splits.

🔧 Troubleshooting

Common Issues and Solutions

Issue: ImportError for packages

Solution:

pip install --upgrade pip pip install -r requirements.txt

Issue: Jupyter kernel not found

Solution:

python -m ipykernel install --user --name=venv

Issue: Docker not running (for Grafana)

Solution:

# macOS open -a Docker # Linux sudo systemctl start docker # Windows # Start Docker Desktop from Start Menu

Issue: Port already in use

Solution:

# Find process using port lsof -i :8888 # For Jupyter lsof -i :3000 # For Grafana # Kill process kill -9 [PID] # Or use different port jupyter lab --port=8889
